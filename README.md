## What is Markdown?
Markdown is a lightweight markup language used to format plain text.
It allows developers to create structured documents easily using simple syntax.
Markdown is commonly used for documentation and README files on GitHub.

## Git Commands Reference

### Check the status of a repository
```bash
git status
````

### Add files to stage

Add one file:

```bash
git add README.md
```

Add all files:

```bash
git add .
```

### Commit changes with a message

```bash
git commit -m "Commit message describing the changes"
```

### Upload changes to the remote repository

```bash
git push origin main
```

### Branch management

Create a branch:

```bash
git branch new-feature
```

Switch to a branch:

```bash
git checkout new-feature
```

Delete a branch:

```bash
git branch -d new-feature
```

### Roll back to a specific commit

View commit history:

```bash
git log
```

Reset to a specific commit:

```bash
git reset --hard COMMIT_ID
```
